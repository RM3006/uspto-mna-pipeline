# USPTO M&A Signal Monitor

## Overview
A data engineering pipeline analyzing USPTO Patent Assignment XML data to detect Intellectual Property transfer trends and M&A signals in the tech sector.

## Tech Stack
- **Ingestion:** Python, AWS S3
- **Warehousing:** Snowflake (Snowpipe, VARIANT, Streams)
- **Transformation:** dbt Core
- **Orchestration:** GitHub Actions
- **Viz:** Power BI
